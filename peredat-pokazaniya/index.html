Передать показания

25 августа 2019 г.

Как передать показания счетчика? Если до недавнего времени показания снимали исключительно сотрудники компаний по поставке газа, то сегодня абоненты сами могут осуществить эту процедуру несколькими способами.

Абоненты обязаны вносить соответствующую оплату до 17 числа каждого месяца. Как передать показания без регистрации, и какие способы оплаты существуют для зарегистрированных абонентов?

Показания счетчиков газа для незарегистрированных абонентов можно передать нарочным способом. При этом следует заполнить необходимую информацию о показаниях прибора учета газа в группе расчета за газ по месту прописки. Помимо этого, абоненты могут позвонить на горячую линию с 12 по 17 число для передачи показаний.

Зарегистрированные абоненты могут передать показания следующими способами:

  • Оплата онлайн. Для этого используйте логин, пароль, номер лицевого счета и полученный шифр.
  • Смс-сообщением по телефону.

Оплата за газ производится максимально просто и комфортно для каждого абонента.
